Main Navigation

CSS selector

.mainNav{}

description :

The Main Nav puts the main function of the company at highlight. That is write a Letter. For Advanced users, who have written a letter and want to track it, other navigation Options are provide. The navigation is done from the top of the website with the use of traditional header nav buttons.

Product Listing

description :

Product rotate in a merry go round fashion style, with the one being chosen at the front. Users can change the view of the product to typical tiled if they want to. The product at the center appears large, has its title beaneath it.

Color Swatches

description :

The Color Swatches uses a dual tone approach. Where Two tones of almost the same hue is used to create a feeling of subtle vibrancy. Also the Duotone can be used as a styling element to create a modern look to the webpage.

Grids

description :

The Grid used by webpage focuses its visual weight on the center, and slowly disperses on the outside. The grid is 5 columns based, and each column gets smaller as it gets farther from the center.
